Visão Geral do Projeto

Projeto idealizado para facilitar a organização dos pátios de motos da Mottu, a ideia principal do dispositivo IoT é através de uma interface o operador da Mottu pode acionar um botão que faz o dispositivo IoT disparar um aviso sonoro por 1,5s e acender um led, para que com isso o operador identifique a moto procurada.

A integração da solução parte do front-end onde será enviada uma requisição para acionar o botão, além do banco de dados registrar cada moto junto com seu dispositivo único, para que tenha exatamente a relação moto-dispositivo, sendo acionado o dispositivo correto.
